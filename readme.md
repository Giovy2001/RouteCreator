# 🧗 Chalk & Track  
Web app semi-ufficiale di **PalaEnrosadira**

Chalk & Track è una web application pensata per palestre di arrampicata indoor, progettata per tracciare, esplorare e condividere blocchi di boulder.  
Nasce come strumento semplice per segnare i blocchi completati, ma si è evoluta in una piattaforma completa per la community della palestra.

---

## 🚀 Funzionalità

### 👤 Utenti
- Profilo personale
- Storico blocchi completati e tracciati
- Sistema a punti basato sull’attività

### 🧩 Blocchi di Boulder
- Creazione nuovi blocchi
- Selezione prese su pannello
- Classificazione prese (tipologia)
- Inserimento beta (istruzioni per la risoluzione)
- Visualizzazione dettagli blocco

### 🔍 Esplora
- Feed dei blocchi recenti
- Contenuti in evidenza
- Scoperta di nuovi problemi da provare
